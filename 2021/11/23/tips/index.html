# #多媒体￬￬￬￬￬

本功能基于 Hexo Tag 功能，使用 media 标签，目前可选择两种类型，即 audio 和 video 。

#音乐￬￬
{% media audio %}
- title: 粤语
  list:
    - https://music.163.com/#/playlist?id=6764091275
{% endmedia %}

#视频￬￬
{% media video %}
- name: "视频1"
  url: https://xxxxxxxxxxxxx.mp4#t=0
{% endmedia %}
